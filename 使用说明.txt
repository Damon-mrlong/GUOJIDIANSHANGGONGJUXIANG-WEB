1. 软件介绍
================================================================================

国际电商工具箱是一款专为电商财务与物流管理设计的辅助工具，整合了以下功能：

  ✓ 财务对账自动化
  ✓ 快递费用计提处理
  ✓ 物流报价智能计算
  ✓ 台账上传模板生成（开发中）

主要特点：
  • 自动化处理，减少人工操作
  • 标准化输出格式
  • 工作区管理，多项目隔离
  • 操作日志实时显示

2. 环境要求与安装
================================================================================

【系统要求】
  ✓ 操作系统：Windows 10 或 Windows 11（不支持Windows 7及以下）
  ✓ 内存：建议4GB以上
  ✓ 磁盘空间：至少500MB可用空间

【必备运行环境】
  ⚠️ 重要：如果双击.exe文件没有反应，请先安装以下组件

  1. Microsoft Visual C++ Redistributable 2015-2022
     下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe
     说明：大部分电脑已自带，如未安装请下载安装

  2. .NET Framework 4.7.2 或更高版本
     说明：Windows 10/11通常已自带

【首次安装步骤】
  1. 将"国际电商工具箱.exe"解压到任意文件夹（建议：D:\工具\）
  
  2. 右键点击.exe文件 → 属性
     - 如果底部有"解除锁定"选项，请勾选并点击"确定"
     - 这是Windows安全机制，解除后即可正常运行
  
  3. 双击运行"国际电商工具箱.exe"
     - 如果仍无法运行，请以管理员身份运行
     - 如果被杀毒软件拦截，请添加到信任列表

【启动问题排查】
  问题：双击.exe没有任何反应
  解决步骤：
    Step 1：确认已安装VC++ Redistributable（见上方必备环境）
    Step 2：在.exe所在文件夹，按住Shift+右键，选择"在此处打开PowerShell窗口"
            输入：.\国际电商工具箱.exe
            查看是否有错误提示
    Step 3：检查杀毒软件是否拦截（查看杀毒软件历史记录）
    Step 4：尝试以管理员身份运行
    Step 5：如仍无法解决，联系技术支持并提供错误信息

3. 首次使用配置
================================================================================

【步骤1】双击运行"国际电商工具箱.exe"

【步骤2】首次启动会弹出欢迎提示，点击"去选择"

【步骤3】选择一个文件夹作为工作区
  说明：工作区是存放所有输入输出文件的根目录
  建议：选择或新建一个专用文件夹，如"D:\电商工具工作区"

【步骤4】系统会自动创建以下标准目录结构：
  工作区/
  ├── 计提实际账单核对/
  │   ├── 计提台账文件夹/（计提账单输入）
  │   └── 实际台账文件夹/（实际账单输入）
  ├── 快递计提台账处理/（放金掌柜计提用表格）
  ├── 规则文件/（放各模块规则文件）
  ├── OMS上传台账模板生成/（预留）
  └── 输出汇总文件夹/（所有处理结果统一输出到这里）

【步骤5】将相关的Excel文件放入对应的输入文件夹

提示：右上角显示"工作区: XXX"表示工作区已加载成功

3. 功能模块说明
================================================================================

┌────────────────────────────────────────────────────────────────────────┐
│ 模块一：资金对账                                                        │
└────────────────────────────────────────────────────────────────────────┘

【功能】自动比对计提账单与实际账单，生成对账报告

【操作步骤】
  1. 准备文件：
     - 计提账单文件 → 放入"财务对账文件夹/Inputs_Accrual/"
     - 实际账单文件 → 放入"财务对账文件夹/Inputs_Actual/"
     - 映射表文件（如有）→ 放入"财务对账文件夹/"

  2. 点击左侧导航"资金对账"

  3. 系统会自动尝试识别输入文件夹中的文件
     - 如未自动识别，点击"选择文件夹/文件"手动选择

  4. 确认路径无误后，点击"开始对账"

  5. 查看底部日志，等待处理完成

  6. 完成后会自动打开"输出汇总文件夹"，查看生成的对账结果文件

【输出文件】
  - 文件名格式：对账结果_YYYYMMDD_HHMMSS.xlsx
  - 包含：匹配明细、未匹配项、差异分析等工作表

┌────────────────────────────────────────────────────────────────────────┐
│ 模块二：快递计提                                                        │
└────────────────────────────────────────────────────────────────────────┘

【功能】处理快递物流账单，提取关键信息并格式化输出

【操作步骤】
  1. 将快递对账单文件（Excel格式）放入"快递计提文件夹/"

  2. 点击左侧导航"快递计提"

  3. 系统会自动识别文件夹中的Excel文件
     - 如未识别，点击"选择源文件"手动选择

  4. 确认文件路径后，点击"开始处理"

  5. 处理完成后自动打开输出文件夹

【输出文件】
  - 文件名格式：快递计提_YYYYMMDD_HHMMSS.xlsx
  - 按标准格式整理的计提清单

┌────────────────────────────────────────────────────────────────────────┐
│ 模块三：报价计算                                                        │
└────────────────────────────────────────────────────────────────────────┘

【功能】根据品牌、目的地、重量、托板数快速计算物流报价

【操作步骤】
  1. 准备报价规则文件（Excel格式，包含品牌、目的地、价格等信息）

  2. 点击左侧导航"报价计算"

  3. 首次使用需加载规则：
     方法A：自动加载
       - 将规则文件命名为"报价规则.xlsx"或"pricing_rules.xlsx"
       - 放入"报价计算文件夹/"
       - 切换工作区或刷新后自动加载

     方法B：手动加载
       - 点击右上角"加载规则"按钮
       - 选择规则文件

  4. 规则加载成功后：
     a) 在"搜索品牌"框输入关键字筛选
     b) 或直接在"选择品牌"下拉框选择
     c) 选择"目的地仓库"
     d) 输入"重量(KG)"和"托板数"

  5. 点击"计算报价"

  6. 查看计算结果：
     - 显示多种物流渠道的价格
     - 绿色高亮显示最优方案
     - 显示详细价格组成

【规则文件格式要求】
  必须包含的列：
    - 品牌（或Brand）
    - 目的地（或Destination）
    - 基础价格相关字段
    - 重量阶梯、托板费用等

┌────────────────────────────────────────────────────────────────────────┐
│ 模块四：上传模版（开发中）                                              │
└────────────────────────────────────────────────────────────────────────┘

【状态】待台账上传功能完善后开发

【说明】此功能将用于生成OMS系统所需的上传台账模板

4. 操作注意事项
================================================================================

【文件格式要求】
  ✓ 所有输入文件必须是Excel格式（.xlsx 或 .xls）
  ✓ 文件不能处于打开状态（请先关闭Excel）
  ✓ 文件名不要包含特殊字符（建议使用中文、英文、数字、下划线）

【工作区管理】
  ✓ 一个工作区对应一个业务/项目
  ✓ 可通过右上角"切换工作区"按钮更换工作区
  ✓ 工作区路径会自动保存，下次启动自动加载
  ✓ 不同工作区的文件和配置完全独立

【输出文件】
  ✓ 所有处理结果统一保存在"输出汇总文件夹/"
  ✓ 文件名自动包含日期时间戳，不会覆盖旧文件
  ✓ 处理完成后系统会自动打开输出文件夹

【日志查看】
  ✓ 底部"实时日志"区域显示所有操作过程
  ✓ 绿色 = 成功信息
  ✓ 蓝色 = 进度信息
  ✓ 橙色 = 警告信息
  ✓ 红色 = 错误信息

【性能建议】
  ✓ 单个Excel文件建议不超过50MB
  ✓ 数据行数建议不超过10万行
  ✓ 处理大文件时请耐心等待，不要重复点击按钮

5. 常见问题解答
================================================================================

Q1：为什么点击"开始处理"没有反应？
A1：请检查：
    1. 是否已配置工作区（右上角显示绿色勾）
    2. 输入文件路径是否正确
    3. 输入文件是否已关闭（不能在Excel中打开）
    4. 查看底部日志是否有错误提示

Q2：处理失败，提示"找不到文件"？
A2：可能原因：
    1. 文件路径包含特殊字符
    2. 文件被移动或删除
    3. 文件权限不足
    建议：重新选择文件，或将文件移到简单路径（如D:\temp\）

Q3：如何更换工作区？
A3：点击右上角的"文件夹"图标，或点击工作区状态标签

Q4：输出文件在哪里？
A4：统一在 工作区\输出汇总文件夹\ 目录下
    处理完成会自动打开该文件夹

Q5：报价计算无法选择品牌？
A5：需要先加载报价规则文件：
    1. 点击右上角"加载规则"按钮
    2. 选择规则Excel文件
    3. 等待加载完成

Q6：能否同时处理多个文件？
A6：目前各功能模块都是单文件处理模式
    需要处理多个文件请依次操作

Q7：软件可以在其他电脑上使用吗？
A7：可以，直接复制.exe文件到其他Windows电脑即可运行
    首次使用需要重新配置工作区

Q8：配置和数据会丢失吗？
A8：不会，配置文件保存在系统用户目录
    工作区文件保存在您选择的文件夹
    只要不删除这些文件，配置和数据都会保留

Q9：规则文件在哪里？如何配置？
A9：规则文件需要手动放置到对应文件夹：
    1. 财务对账：映射表文件放入"财务对账文件夹/"
    2. 报价计算：报价规则文件放入"报价计算文件夹/"
    建议：首次使用时，将压缩包中附带的示例规则文件复制到相应文件夹
    规则文件通常是Excel格式，可根据实际需要修改

Q10：双击.exe没反应，怎么办？
A10：90%的原因是缺少VC++ Redistributable运行库
     解决方法：
     1. 下载安装：https://aka.ms/vs/17/release/vc_redist.x64.exe
     2. 安装完成后重启电脑
     3. 再次尝试运行.exe
     详细排查步骤见上方"启动问题排查"章节

Q11：遇到其他问题怎么办？
A11：请联系技术支持，并提供：
    1. 底部日志的完整截图
    2. 问题复现的详细步骤
    3. 输入文件的示例（隐私信息可脱敏）
